# Titanic Survival Prediction

A complete end-to-end Machine Learning project predicting passenger survival on the Titanic.
Three models, hyperparameter tuning, rich visualisations, and a Kaggle-ready submission.

## Quickstart

### Step 1 — Download the Titanic Dataset

1. Go to: https://www.kaggle.com/competitions/titanic/data
2. Sign in or create a free Kaggle account
3. Click **"Download All"** to get a zip file
4. Extract and place **train.csv** and **test.csv** into the `data/` folder

**Alternative (Kaggle CLI):**
```bash
pip install kaggle
# Place kaggle.json in ~/.kaggle/
kaggle competitions download -c titanic -p data/
cd data
tar -xf titanic.zip
cd ..
```

---

### Step 2 — Set Up Python Environment

```bash
# Create a virtual environment
python -m venv venv

# Activate on Windows
venv\Scripts\activate

# Activate on macOS / Linux
source venv/bin/activate

# Install all dependencies
pip install -r requirements.txt
```

---

### Step 3 — Run the Pipeline

Run all scripts from the **project root** directory:

```bash
# Step 1: Preprocess data and engineer features
python src/data_preprocessing.py

# Step 2: Train all models with hyperparameter tuning
python src/model_training.py

# Step 3: Generate predictions and submission file
python src/predict.py
```

Your submission file will be at: `outputs/submission.csv`

## Models Trained

| Model               | Tuned         | Typical Accuracy |
|---------------------|---------------|-----------------|
| Logistic Regression | GridSearchCV  | ~80%            |
| Random Forest       | GridSearchCV  | ~82%            |
| XGBoost             | GridSearchCV  | ~83%            |

The best model by ROC-AUC is automatically selected and saved as `best_model.pkl`.

## Feature Engineering

| Feature    | Description                                      |
|------------|--------------------------------------------------|
| Title      | Extracted from Name (Mr, Mrs, Miss, Master, Rare)|
| FamilySize | SibSp + Parch + 1                                |
| IsAlone    | 1 if travelling solo                             |
| AgeGroup   | Child / Teen / YoungAdult / Adult / Senior       |
| FareGroup  | Quartile bins: Low / Mid / High / VeryHigh       |
| HasCabin   | 1 if cabin number is present (proxy for wealth)  |

## Troubleshooting

| Problem                         | Fix                                                              |
|---------------------------------|------------------------------------------------------------------|
| FileNotFoundError: train.csv    | Place train.csv and test.csv in the data/ folder                 |
| ModuleNotFoundError: xgboost    | Run pip install -r requirements.txt                              |
| Scripts fail to find modules    | Run scripts from the project root, not from inside src/          |
| Out of memory during GridSearch | Reduce n_estimators or max_depth values in model_training.py     |
